# Python Industrial – Automation & Control Logic

Industrial Python scripts focused on automation, welding sequences, repetitive motion, sensor validation, and basic safety logic for cobot and robotic environments.

---

## Overview (English)

This repository contains a collection of Python scripts designed for industrial automation scenarios.  
The goal is to demonstrate clear, readable, and practical logic used in real manufacturing environments, especially in cobot cells, robotic welding stations, and automated inspection systems.

The scripts are intentionally short and easy to understand, making them ideal for recruiters, technical interviews, and industrial engineering teams evaluating automation skills.

---

## Included Modules

- Welding Sequences — MIG/MAG welding passes, arc activation, linear motion simulation.  
- Repetitive Motion — cycles, position transitions, movement loops used in cobot routines.  
- Sensor Validation — boolean checks, process gating, interlock logic.  
- Safety Logic — emergency stop routines, fail-safe conditions, system validation.  
- Control Logic — simple decision-making structures used in industrial automation.

---

## How to Run

All scripts are standalone.  
Example:

python welding_sequence.py

# Python Industrial – Automatización y Lógica de Control

Scripts de Python orientados a automatización industrial: secuencias de soldadura, movimiento repetitivo, validación de sensores y lógica básica de seguridad para entornos de cobots y robótica.

---

## Descripción General

Este repositorio contiene scripts diseñados para escenarios de automatización industrial.  
El objetivo es mostrar lógica clara, legible y práctica utilizada en entornos reales de manufactura, especialmente en celdas de cobots, estaciones de soldadura robotizada y sistemas de inspección automatizada.

Los scripts son cortos y fáciles de entender, ideales para reclutadores, entrevistas técnicas y equipos de ingeniería industrial.

---

## Módulos Incluidos

- Secuencias de Soldadura — pasadas MIG/MAG, activación del arco, simulación de movimiento lineal.  
- Movimiento Repetitivo — ciclos, transiciones de posición, bucles de movimiento.  
- Validación de Sensores — chequeos booleanos, lógica de interbloqueo, habilitación de procesos.  
- Lógica de Seguridad — rutinas de paro de emergencia, condiciones de seguridad, validación del sistema.  
- Lógica de Control — estructuras simples de decisión usadas en automatización industrial.
